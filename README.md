# 🏭 鑫胜隆废钢再生资源实时数据跟踪分析系统

## 系统简介

专为废钢再生资源行业设计的实时数据跟踪分析系统，支持采购、出库、费用、库存、利润的全面管理。

## 核心功能

### 📊 统计看板
- 关键指标卡片（采购量、出库量、库存、利润）
- 月度趋势图（采购/出库量 + 利润走势）
- 料型进出对比图
- 供应商/收货单位 TOP10
- 费用支出分析（饼图 + 趋势）

### 📝 数据录入
- 采购记录录入（供应商、料型、重量、价格、运费、扣杂）
- 出库记录录入（收货单位、料型、重量、钢厂扣杂、运费）
- 费用记录录入（科目、金额、经手人）

### 💰 成本利润分析
- 各料型采购成本分析
- 各料型销售均价对比
- 月度利润构成图
- 料型毛利排名
- 费用明细透视表

### 📦 库存管理
- 上月余量录入
- 各料型库存实时计算（期初 + 入库 - 出库）
- 库存预警标记
- 库存分布饼图

### 💳 付款跟踪
- 供应商应付账款明细
- 收货单位应收账款统计
- 未付款标记（时间/经手人/金额）

### 📤 报表导出
- 采购/出库/费用台账 CSV 导出
- 月度汇总报表
- 料型分析报表

## 技术特性

- 📱 **手机端适配**：响应式布局，手机浏览器直接访问
- 📈 **实时图表**：Plotly 交互式图表，支持缩放
- 💾 **数据兼容**：直接使用现有 Excel 台账文件
- 🔄 **实时更新**：数据录入后立即反映在图表中

## 安装与运行

### 方式一：本地运行（推荐）

#### 1. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

#### 2. 启动应用

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
bash start.sh
```

**或直接:**
```bash
streamlit run app.py
```

#### 3. 访问应用

浏览器打开: http://localhost:8501

同一局域网内手机访问: http://<电脑IP>:8501

### 方式二：部署到 Streamlit Cloud（互联网访问）

1. 将项目推送到 GitHub 仓库

3. 连接 GitHub 仓库部署
4. 获得公开 URL，手机随时随地访问

### 方式三：部署到自有服务器

```bash

```

```bash

python -m streamlit run app.py --server.port 8501 --server.address 0.0.0.0
```

## 数据文件说明

将您的台账文件放入 `data/` 目录：

| 文件 | 说明 |
|------|------|
| `data/purchase.xlsx` | 采购台账（多 sheet，每月一 sheet） |
| `data/sales.xlsx` | 出库台账（多 sheet，每月一 sheet） |
| `data/expenses.xlsx` | 费用报销台账（多 sheet，每月一 sheet） |

系统会自动识别 sheet 并按月份汇总。

## 料型分类

系统自动识别以下料型：
三级原料、二级原料、统四、油漆桶、花盒子、花盒子压块、三级压块、
电脑壳子、破碎料、火烧铁料、洋钉、弹簧床、铁皮彩瓦、钢筋料、镀锌料、
重废、钢架子、自行车架子、铁丝、圆管、方管、围栏、岩棉接板、尾料等

## 费用分类

自动分类为：工资、电费、油费、食堂餐费、修理费、差旅费、招待费、
办公费、运输费、福利费、税费、低值易耗品

## 计算逻辑

### 采购成本
- 实际净重 = 净重 - 扣杂
- 采购成本 = 扣杂后金额 + 运费 = 实付金额

### 销售收入
- 到手金额 = 销售金额 - 运费 - 罚款 - 手续费

### 利润
- 毛利 = 到手金额 - 采购成本
- 净利润 = 毛利 - 固定成本（费用）

### 库存
- 期末库存 = 上月余量 + 本期采购 - 本期出库（按料型）
